## 🔧 技术方案

### 方案1：修复 IndexedDB Mock（推荐）

**问题**: 手动 mock 不够完整，不支持异步操作

**解决方案**: 使用 fake-indexeddb 库

```bash
pnpm add -D fake-indexeddb
```

**优势**:
- 完整的 IndexedDB 实现
- 支持异步操作
- 更接近真实环境

---

### 方案2：配置 Coverage 工具

**问题**: @vitest/coverage-v8@4.1.0 版本不兼容

**解决方案**: 降级到兼容版本

```bash
pnpm remove @vitest/coverage-v8
pnpm add -D @vitest/coverage-v8@1.6.0
```

---

### 方案3：测试隔离

**问题**: 部分测试相互影响

**解决方案**:
1. 使用 describe.each 进行数据驱动测试
2. 使用 beforeEach/afterEach 清理状态
3. 使用 vi.clearAllMocks()
