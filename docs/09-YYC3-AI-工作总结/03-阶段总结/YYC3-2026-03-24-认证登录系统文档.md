# YYC³ 认证登录系统使用文档

## 📖 概述

YYC³ 认证登录系统提供完整的用户认证功能，包括登录、注册、忘记密码和密码重置。该系统使用 React Context 进行状态管理，支持本地存储持久化。

## 🚀 快速开始

### 1. 安装依赖

确保项目已安装以下依赖：

```bash
pnpm install react-router-dom
pnpm install lucide-react
```

### 2. 集成到应用

#### 方式一：使用 AppAuthProvider（推荐）

```tsx
// src/App.tsx
import React from 'react';
import { AppAuthProvider } from './providers/AuthProvider';

function App() {
  return (
    <AppAuthProvider />
  );
}

export default App;
```

#### 方式二：手动集成

```tsx
// src/App.tsx
import React from 'react';
import { BrowserRouter } from 'react-router-dom';
import { AuthProvider } from './contexts/AuthContext';
import { AuthRoutes } from './pages/auth-routes';

function App() {
  return (
    <BrowserRouter>
      <AuthProvider>
        <AuthRoutes />
      </AuthProvider>
    </BrowserRouter>
  );
}

export default App;
```

## 🔧 使用示例

### 1. 使用 useAuth Hook

```tsx
import { useAuth } from '../contexts/AuthContext';

function MyComponent() {
  const { user, isAuthenticated, login, logout } = useAuth();

  if (!isAuthenticated) {
    return <div>请先登录</div>;
  }

  return (
    <div>
      <h1>欢迎, {user?.username}</h1>
      <button onClick={logout}>退出登录</button>
    </div>
  );
}
```

### 2. 使用 ProtectedRoute 保护路由

```tsx
import { ProtectedRoute } from '../contexts/AuthContext';

function MyProtectedComponent() {
  return (
    <ProtectedRoute fallback={<LoginPage />}>
      <div>这是受保护的内容</div>
    </ProtectedRoute>
  );
}
```

### 3. 登录示例

```tsx
import { useAuth } from '../contexts/AuthContext';

function LoginForm() {
  const { login, isLoading } = useAuth();
  const [formData, setFormData] = useState({
    email: '',
    password: ''
  });

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    
    const response = await login(formData);
    
    if (response.success) {
      console.log('登录成功', response.user);
      // 跳转到首页
      navigate('/');
    } else {
      console.error('登录失败', response.error);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      {/* 表单字段 */}
      <button type="submit" disabled={isLoading}>
        {isLoading ? '登录中...' : '登录'}
      </button>
    </form>
  );
}
```

## 📝 API 参考

### AuthContext 类型

```typescript
interface AuthContextType {
  user: User | null;              // 当前用户信息
  status: AuthStatus;              // 认证状态
  isLoading: boolean;               // 是否正在加载
  isAuthenticated: boolean;         // 是否已认证
  
  // 方法
  login(data: LoginFormData): Promise<AuthResponse>;
  logout(): Promise<void>;
  register(data: RegisterFormData): Promise<AuthResponse>;
  forgotPassword(data: ForgotPasswordFormData): Promise<AuthResponse>;
  resetPassword(data: ResetPasswordFormData): Promise<AuthResponse>;
  updateUser(data: Partial<User>): Promise<void>;
  refreshUser(): Promise<void>;
}
```

### AuthStatus 枚举

```typescript
type AuthStatus = 'loading' | 'authenticated' | 'unauthenticated' | 'error';
```

### User 类型

```typescript
interface User {
  id: string;
  email: string;
  username: string;
  avatar?: string;
  role: 'admin' | 'user' | 'guest';
  createdAt: string;
  updatedAt: string;
  lastLoginAt?: string;
}
```

## 🔐 安全说明

### 当前实现（演示版）

- ✅ 使用 localStorage 存储用户信息
- ✅ 模拟 API 调用
- ✅ 基本的前端验证
- ⚠️ Token 为模拟的 JWT，不包含加密

### 生产环境建议

在生产环境中，建议：

1. **后端 API 集成**
   ```typescript
   const login = async (data: LoginFormData) => {
     const response = await fetch('/api/auth/login', {
       method: 'POST',
       headers: { 'Content-Type': 'application/json' },
       body: JSON.stringify(data),
     });
     
     const result = await response.json();
     // 处理结果...
   };
   ```

2. **Token 安全存储**
   - 使用 httpOnly cookies 存储 token
   - 或使用安全的 localStorage（配合 CSRF 保护）

3. **密码加密**
   - 在发送到服务器前使用 bcrypt 等算法加密

4. **添加刷新 Token 机制**
   - 实现 Token 自动刷新
   - 处理 Token 过期

## 🧪 测试

### 单元测试示例

```typescript
import { renderHook, act } from '@testing-library/react';
import { AuthProvider, useAuth } from '../contexts/AuthContext';

describe('useAuth', () => {
  it('should login successfully', async () => {
    const wrapper = ({ children }) => (
      <AuthProvider>{children}</AuthProvider>
    );
    
    const { result } = renderHook(() => useAuth(), { wrapper });
    
    await act(async () => {
      const response = await result.current.login({
        email: 'demo@example.com',
        password: 'password123'
      });
      
      expect(response.success).toBe(true);
      expect(result.current.isAuthenticated).toBe(true);
    });
  });
});
```
