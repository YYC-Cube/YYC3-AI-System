## 📈 性能指标

### Web Vitals标准
| 指标 | 警告阈值 | 临界阈值 | 说明 |
|------|----------|----------|------|
| FCP (First Contentful Paint) | 1800ms | 3000ms | 首次内容绘制 |
| LCP (Largest Contentful Paint) | 2500ms | 4000ms | 最大内容绘制 |
| FID (First Input Delay) | 100ms | 300ms | 首次输入延迟 |
| CLS (Cumulative Layout Shift) | 0.1 | 0.25 | 累积布局偏移 |
| TTFB (Time to First Byte) | 600ms | 800ms | 首字节时间 |
| Page Load Time | 2500ms | 3500ms | 页面加载时间 |
| Frame Rate | 50fps | 30fps | 帧率 |

### 性能分数计算
- **FCP分数**: Math.max(0, 100 - (FCP / 1800) * 100)
- **LCP分数**: Math.max(0, 100 - (LCP / 2500) * 100)
- **FID分数**: Math.max(0, 100 - (FID / 100) * 100)
- **CLS分数**: Math.max(0, 100 - (CLS / 0.1) * 100)
- **加载分数**: Math.max(0, 100 - (PageLoad / 2500) * 100)
- **综合分数**: (FCP + LCP + FID + CLS + 加载) / 5
