## 5. DnD 面板拖拽交换系统

### 5.1 系统概述

**目的**: 允许用户通过拖拽来交换三栏面板的内容位置
**技术**: `react-dnd` + `HTML5Backend`
**持久化**: panelMap 存储在 Zustand + persist (localStorage)

### 5.2 DnD 类型定义

```typescript
const PANEL_DND_TYPE = 'PANEL_SWAP';
type PanelSlot = 'left' | 'middle' | 'right';

interface DragItem {
  type: typeof PANEL_DND_TYPE;
  slot: PanelSlot;
}
```

### 5.3 DragHandle 组件

```typescript
function DragHandle({ slot, isDark }: { slot: PanelSlot; isDark: boolean }) {
  const [{ isDragging }, drag] = useDrag(() => ({
    type: PANEL_DND_TYPE,
    item: { type: PANEL_DND_TYPE, slot } as DragItem,
    collect: (monitor) => ({
      isDragging: monitor.isDragging(),
    }),
  }), [slot])

  return (
    <div ref={drag} className="cursor-grab active:cursor-grabbing p-0.5 rounded ...">
      <GripVertical className="w-3.5 h-3.5" />
    </div>
  )
}
```

**视觉状态**:

| 状态     | 样式                                                     |
| -------- | -------------------------------------------------------- |
| 默认(暗) | `text-white/15 hover:text-white/40 hover:bg-white/5`     |
| 默认(亮) | `text-slate-300 hover:text-slate-500 hover:bg-slate-100` |
| 拖拽中   | `opacity-50 scale-90`                                    |
| 光标     | `cursor-grab` / `active:cursor-grabbing`                 |

### 5.4 DropZone 组件

```typescript
function DropZone({ slot, onSwap, children, isDark }) {
  const [{ isOver, canDrop }, drop] = useDrop(() => ({
    accept: PANEL_DND_TYPE,
    drop: (item: DragItem) => {
      if (item.slot !== slot) onSwap(item.slot, slot)
    },
    canDrop: (item: DragItem) => item.slot !== slot,
    collect: (monitor) => ({
      isOver: monitor.isOver(),
      canDrop: monitor.canDrop(),
    }),
  }), [slot, onSwap])

  return (
    <div ref={drop} className="relative h-full">
      {children}
      {/* 拖入叠加层 */}
      {isOver && canDrop && (
        <div className="absolute inset-0 z-30 pointer-events-none rounded-xl
          border-2 border-dashed border-indigo-400/50
          bg-indigo-500/[0.08] backdrop-blur-[1px]
          flex items-center justify-center">
          <div className="px-4 py-2 rounded-xl ... text-[12px]">
            释放以交换面板
          </div>
        </div>
      )}
      {/* 可放置指示 (拖拽中但未悬停) */}
      {!isOver && canDrop && (
        <div className="absolute inset-0 z-30 pointer-events-none
          rounded-lg border border-dashed border-indigo-400/20" />
      )}
    </div>
  )
}
```

**叠加层视觉状态**:

| 状态             | 描述                     | 视觉效果                                        |
| ---------------- | ------------------------ | ----------------------------------------------- |
| 空闲             | 无拖拽发生               | 无叠加层                                        |
| canDrop          | 正在拖拽，但未悬停此区域 | 虚线边框 (`border-dashed border-indigo-400/20`) |
| isOver + canDrop | 悬停在此区域上方         | 蓝色半透明叠加 + "释放以交换面板" 文字          |
| 自身             | 拖到自己的位置           | 不响应 (`canDrop: item.slot !== slot`)          |

```typescript
  const contentType = panelMap[slot]           // 从 Store 读取当前映射
  const Content = CONTENT_COMPONENTS[contentType]
  const Toolbar = TOOLBAR_COMPONENTS[contentType]
  return (
    <DropZone slot={slot} onSwap={handlePanelSwap} isDark={t.isDark}>
      <div className="flex flex-col h-full">
        <div className="flex items-center">
          <DragHandle slot={slot} isDark={t.isDark} />
          <div className="flex-1 min-w-0">
            <Toolbar />
          </div>
        </div>
        <div className="flex-1 overflow-hidden">
          <Content />
        </div>
      </div>
    </DropZone>
  )
}
```

### 5.7 交换持久化

```typescript
// Store 中的 panelMap
panelMap: { left: 'chat', middle: 'files', right: 'code' }

// 交换函数
swapPanels: (from: PanelSlot, to: PanelSlot) => {
  const next = { ...state.panelMap }
  const temp = next[from]
  next[from] = next[to]
  next[to] = temp
  return { panelMap: next }
}
```

**持久化**: `panelMap` 通过 Zustand `persist` 中间件写入 `localStorage['yyc3-storage']`

### 5.8 交换流程图

```
用户按住 DragHandle (如 left 栏的 GripVertical)
    |
    v
useDrag 触发: item = { type: 'PANEL_SWAP', slot: 'left' }
    |
    v
拖拽中: DropZone(middle) 和 DropZone(right) 显示虚线边框
    |
    v
悬停在 DropZone(right) 上: 显示蓝色叠加 + "释放以交换面板"
    |
    v
释放鼠标: useDrop.drop() 触发
    |
    v
onSwap('left', 'right') → swapPanels('left', 'right')
    |
    v
panelMap: { left: 'code', middle: 'files', right: 'chat' }
    |
    v
renderPanel('left') 现在渲染 CodeEditor
renderPanel('right') 现在渲染 ChatInterface
    |
```

## 12. 性能优化

### 12.1 渲染优化

| 策略                 | 实现                                        | 说明                 |
| -------------------- | ------------------------------------------- | -------------------- |
| useCallback          | `handlePanelSwap`                           | 避免 DropZone 重渲染 |
| useCallback          | `handleKeyDown`                             | 稳定事件引用         |
| useDrag/useDrop memo | `[slot]` 依赖                               | 仅 slot 变更时重创建 |

### 12.2 协同性能

| 策略           | 实现                    | 说明                        |
| -------------- | ----------------------- | --------------------------- |
| 定时同步       | 2000ms interval         | 非实时，减少 Store 更新频率 |
| Awareness 事件 | 增量更新                | 仅状态变更时触发            |
| 过滤本地用户   | `u.id !== 'local-user'` | 避免自身数据进入协同列表    |

### 12.3 布局性能

| 策略                | 实现            | 说明                 |
